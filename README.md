# AI Meme and Poster Generator

## Project Description
This project is a web-based AI application that generates memes and posters automatically based on user input. The system creates captions using an AI-based text generation module and renders them on poster templates using image processing techniques.

---

## Problem Statement
Students often require posters and memes for college events, social media, and academic activities. Designing such content manually requires creativity and design skills. This project aims to simplify content creation using AI-driven caption generation and automated poster design.

---

## Objectives
- To build a full-stack web application using AI concepts
- To generate captions automatically based on user input
- To create posters dynamically using image processing
- To provide an easy-to-use interface for students

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- AI Logic: Mock AI-based NLP caption generator
- Image Processing: Pillow
- Tools: VS Code

---

## System Architecture
1. User enters topic and tone on the web interface
2. Frontend sends request to backend API
3. AI module generates a caption
4. Image generator places caption on a poster template
5. Final poster is displayed to the user
